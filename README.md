# stb-test
Simple web application testing using Java Selenium and TestNG

### Tools Used
* java 18.0.1.1
* Selenium 4.29.0
* TestNG 7.10.2

### Project Structure
* src/main/java/models      # Page Objects
* src/main/java/utility     # Utility classes
* src/test/java/             # Test cases


### Run Tests
* mvn test
* mvn test -DsuiteXmlFile=testng.xml
